## Setup execution steps 

- From the `src` directory:
  - **1)** Run the script `build-venv.sh`
  - **2)** Run the comand `source .venv/bin/activate` with `make build-env`
  - **3)** Now inside the virtual environment run the script `init_setup.sh` with `run-setup`
  - **4)** Run the application with the `make run-app`
